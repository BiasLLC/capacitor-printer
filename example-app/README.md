# Example App for `@capgo/capacitor-printer`

This Vite project links directly to the local plugin source so you can exercise the native APIs while developing.

## Actions in this playground

- **Print Base64** – Print files encoded as base64 strings (PDFs, images).
- **Print File** – Print device files using file paths or content URIs.
- **Print HTML** – Print HTML documents with custom styling.
- **Print PDF** – Print PDF documents from device storage.
- **Print WebView** – Print the current web view content.

## Getting started

```bash
npm install
npm start
```

Add native shells with `npx cap add ios` or `npx cap add android` from this folder to try behaviour on device or simulator.
